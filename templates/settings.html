📡 Webhook端点信息

URL: {{ request.url_root }}webhook

方法: POST

Content-Type: application/json

使用示例:

基础请求:
curl -X POST {{ request.url_root }}webhook \
  -H "Content-Type: application/json" \
  -d '{"event": "test", "message": "Hello Webhook"}'
带签名验证:
curl -X POST {{ request.url_root }}webhook \
  -H "Content-Type: application/json" \
  -H "X-Hub-Signature-256: sha256=签名值" \
  -d '{"event": "push", "data": "some data"}'
GitHub风格Webhook:
curl -X POST {{ request.url_root }}webhook \
  -H "Content-Type: application/json" \
  -H "X-Event-Type: push" \
  -H "X-Hub-Signature-256: sha256=签名值" \
  -d '{"action": "opened", "repository": {...}}'

🔒 签名验证说明:

如果设置了密钥，请求需要包含 X-Hub-Signature-256 或 X-Signature 头部，值为 HMAC-SHA256 签名。

签名计算: hmac.new(secret.encode(), payload, hashlib.sha256).hexdigest()
